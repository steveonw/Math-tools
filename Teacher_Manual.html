3. Individual Tool Guides

🚴 01 — Bike Simulator
Calc 1 Derivatives

Purpose

Builds intuition for derivatives as rate of change by visualizing a cyclist's position, velocity, and acceleration in real-time.

Key Features

  • Three synchronized graphs: position s(t), velocity v(t) = s'(t), acceleration a(t) = s''(t)
  • Adjustable speed function parameters
  • Tangent line visualization at any point
  • Animation controls (play, pause, step)

Suggested Use

Introduce before formal derivative definition. Ask students: "When is the bike speeding up? Slowing down? How do you know from each graph?"

🥋 02 — Derivative Dojo v5
Calc 1 Derivative Rules

Purpose

Drill-based practice for all derivative rules with immediate feedback and adaptive difficulty.

Key Features

  • Rule Categories: Power, Product, Quotient, Chain, Trigonometric, Exponential, Logarithmic
  • Boss Rush Mode: Timed challenge combining all rules
  • 100+ problems with step-by-step hints
  • Progress tracking: Belt ranking system (White → Black)
  • Common error detection: Identifies specific mistakes

Suggested Use

Assign specific rule categories after teaching each rule. Use Boss Rush mode for exam review.

🔬 03 — Function Anatomy Lab v3
Calc 1 Curve Sketching

Purpose

Interactive exploration of the relationship between a function and its first and second derivatives.

Key Features

  • Synchronized f(x), f'(x), f''(x) graphs
  • Draggable point showing corresponding values on all three graphs
  • Automatic detection of critical points, inflection points, and extrema
  • Multiple function families: polynomials, rational, trigonometric, exponential
  • Sign analysis overlays

Suggested Use

Project during lecture when introducing curve sketching. Have students predict f' sign from f behavior before revealing.

🏃 04 — Related Rates Shadow Lab v3
Calc 1 Related Rates

Purpose

Visualizes the classic "shadow problem" — a person walking away from a streetlamp — with step-by-step problem-solving guidance.

Key Features

  • Animated scene: person, streetlamp, shadow moving in real-time
  • Adjustable parameters: lamp height, person height, walking speed
  • Step-by-step guided solution (identify variables → draw diagram → write equation → differentiate → solve)
  • Formula overlay showing similar triangles relationship
  • Rate display: dx/dt, ds/dt (shadow tip), dL/dt (shadow length)

Suggested Use

Work through as a class first, then have students change parameters and predict how rates change before running animation.

🔄 05 — Optimization Paradox Lab v2
Calc 1 Optimization

Purpose

Teaches optimization through cognitive conflict — students discover why their intuition fails before learning the calculus solution.

Key Features

  • 6 Classic Problems: Fence by River, Box Volume, Tin Can, Ladder Around Corner, Poster Margins, Rectangle Under Parabola
  • Explore Mode: Interactive sliders showing constraint coupling in real-time
  • Challenge Mode: "Gut choice" buttons with labeled cognitive biases (Symmetry Bias, Endpoint Bias, etc.)
  • Failure feedback: Shows exactly why intuitive answers fail
  • Math Path: 3-step verification (constraint → substitution → derivative)
  • Animated transitions when exploring parameters

Cognitive Biases Taught

  • Symmetry Bias: "Squares are always optimal!"
  • Endpoint Bias: "Maximize one dimension!"
  • Ignoring Constraint Coupling: Treating variables as independent

Suggested Use

Have students try gut choices first (without penalty), discuss as class why each fails, then solve together.

🔒 06 — Integral Lockbox v5
Calc 2 Integration Techniques

Purpose

Gamified integration practice where each technique is a "tumbler" that unlocks a puzzle box.

Key Features

  • 5 Technique Tumblers: U-Substitution, Integration by Parts, Partial Fractions, Trig Substitution, Simplify First
  • 60+ problems across all difficulty levels
  • Pattern Quest mode: Learn to identify which technique to use
  • Decision Tree Wizard: Interactive flowchart for technique selection
  • Star rating system: Track mastery per technique
  • Step-by-step hints available for each problem

Suggested Use

Assign one tumbler per week as techniques are taught. Use Pattern Quest mode before exams to practice technique identification.

🍩 07 — Volumes of Revolution v3
Calc 2 Applications of Integration

Purpose

3D visualization of solids of revolution with disk, washer, and shell methods.

Key Features

  • Three methods: Disk, Washer, Shell — with visual explanation of each
  • 3D rotating view: See the solid from any angle
  • Representative slice animation: Watch a slice sweep to form the solid
  • Formula builder: Shows how integral is constructed
  • Multiple axes: Revolve around x-axis, y-axis, or arbitrary lines
  • Preset examples + custom function input

Suggested Use

Demo during lecture to show why shell method is easier for certain problems. Have students predict volume before calculation.

🚗 08 — Taylor Journey (Close Enough) v0.9
Calc 2 Taylor Series

Purpose

A narrative-driven game where students discover Taylor series through a "business simulation" — finding polynomial approximations that are "close enough" for buyers.

Key Features

  • 6-Phase Discovery Arc:
    1. Zoom & Guess (contracts 1-6)
    2. Tangents Rule (7-10)
    3. Curvature Bites (11-14)
    4. Pain Point (15-18)
    5. Taylor Reveal (19-20) — "Aha!" moment
    6. Aftermath — Duel + Sandbox unlocked
  • Duel Mode: Taylor vs. Least Squares comparison
  • Sandbox Workshop: Full control over function, center point, degree
  • Coefficient breakdown: Shows c₀ = f(a), c₁ = f'(a), etc.
  • Convergence warnings: Displays radius of convergence for each function
  • Error visualization: Shaded remainder region R_n(x)

Teacher Mode

Hold Shift + ~ for 2 seconds to open Teacher panel with options to skip phases, freeze buyer, or unlock all modes.

Suggested Use

Assign as homework before Taylor lecture. Students arrive having "discovered" the pattern. Use Duel mode to compare Taylor vs. least squares fit.

🐢 09 — Zeno's Paradox
Calc 1 Calc 2 Limits & Series

Purpose

Visualizes the Achilles and Tortoise paradox to build intuition for limits and infinite series.

Key Features

  • Animated race between Achilles and the Tortoise
  • Step-by-step visualization: each "catch-up" attempt
  • Running sum display: 1/2 + 1/4 + 1/8 + ... → 1
  • Geometric series formula connection
  • Adjustable parameters (speeds, head start)

Suggested Use

Use on day 1 of limits (Calc 1) or series (Calc 2) to motivate why we need rigorous definitions.

🧭 10 — Polar Explorer
Calc 2 Polar Coordinates

Purpose

Interactive polar coordinate graphing with area calculation.

Key Features

  • Function library: Cardioids, roses, limaçons, spirals, lemniscates
  • Custom equation input: Enter any r = f(θ)
  • Animated drawing: Watch curve trace as θ increases
  • Area calculator: Computes ½∫r²dθ with visual shading
  • Cartesian overlay option
  • Point coordinates display: (r, θ) and (x, y) simultaneously

Suggested Use

Have students predict shape before animation plays. Use area shading to visualize what the integral calculates.

🚀 11 — Vector Void v2
Calc 2/3 Vectors

Purpose

Space navigation game teaching vector operations through gameplay.

Key Features

  • Missions: Navigate through space using vector addition
  • Operations covered: Addition, scalar multiplication, dot product, cross product, projection
  • Visual representation: 2D and 3D vector displays
  • Challenge modes: Timed navigation, angle calculations
  • Geometric interpretation: Shows angle between vectors, parallel/perpendicular detection

Suggested Use

Assign after introducing each operation. Game context makes abstract operations concrete.

4. Classroom Integration Ideas

Before Lecture (Flipped Classroom)

  • Taylor Journey: Assign contracts 1-10 as "discovery homework" — students arrive having found patterns
  • Optimization Paradox: Have students try gut choices at home, discuss failures in class
  • Zeno's Paradox: Watch animation before limits lecture

During Lecture

  • Function Anatomy Lab: Project and drag point across curve while asking "Is f increasing here? Is f' positive?"
  • Volumes of Revolution: Show 3D solid forming in real-time
  • Related Rates: Animate shadow problem while building equation

Homework & Practice

  • Derivative Dojo: "Complete 20 problems in Chain Rule category"
  • Integral Lockbox: "Unlock the U-Substitution tumbler"
  • Polar Explorer: "Graph these 5 equations and sketch by hand to compare"

Exam Review

  • Derivative Dojo Boss Rush: Timed mixed practice
  • Integral Lockbox Pattern Quest: Technique identification drill
  • Optimization Paradox: Review all 6 problem types

Assessment Ideas

  • Have students screenshot their Derivative Dojo belt progression
  • Assign specific Taylor Journey sandbox explorations with written analysis
  • Quiz: "Without the tool, predict which integration technique is needed"

5. Technical Notes

Browser Compatibility

All tools tested and working on:

  • Chrome 90+
  • Firefox 88+
  • Safari 14+
  • Edge 90+

Data Storage

Progress is saved to browser localStorage. This means:

  • Progress persists between sessions on the same device/browser
  • Clearing browser data will reset progress
  • Progress does not sync between devices
  • Each tool has independent storage
⚠️ Note: If students use incognito/private browsing mode, progress will not be saved.

Offline Use

All tools work offline after initial download. The only external resources are Google Fonts (for styling), which will fall back to system fonts if unavailable.

Mobile Support

All tools are responsive and work on tablets. Phone screens are supported but may be cramped for some visualizations. Landscape orientation recommended for mobile use.

Resetting Progress

Most tools include a reset option in settings. Alternatively, students can clear localStorage for the specific tool in browser developer tools.
