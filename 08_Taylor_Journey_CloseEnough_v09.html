Taylor form
T_k(x) = Σ (f^(n)(a)/n!) (x-a)^n
Coefficient breakdown
c₀ = f(a) = ...
c₁ = f'(a) = ...
c₂ = f''(a)/2! = ...
Common Taylor Series (memorize!)
eˣ = 1 + x + x²/2! + x³/3! + ... (all x)
sin x = x − x³/3! + x⁵/5! − ... (all x)
cos x = 1 − x²/2! + x⁴/4! − ... (all x)
ln(1+x) = x − x²/2 + x³/3 − ... (|x| < 1)
1/(1−x) = 1 + x + x² + x³ + ... (|x| < 1)
arctan x = x − x³/3 + x⁵/5 − ... (|x| ≤ 1)
Error Bound (Remainder Theorem)
R_n(x) = f(x) − T_n(x)

Lagrange form:
|R_n(x)| ≤ M·|x−a|n+1 / (n+1)!

where M = max|f(n+1)(c)| for c between a and x

Key insight: Error shrinks with more terms AND when x is closer to a
⚠️ Warning: This function has limited convergence radius!
