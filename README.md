# 🧮 Interactive Calculus Tools Bundle

A collection of 11 interactive HTML tools covering Calculus 1 & 2 topics.
All tools are self-contained single HTML files - just open in any modern browser!

## 📚 Tools Included

| # | Tool | Topic | Key Features |
|---|------|-------|--------------|
| 01 | **Bike Simulator** | Derivatives / Rate of Change | Position → velocity → acceleration visualization |
| 02 | **Derivative Dojo v5** | Derivative Rules | 100+ problems, power/chain/product/quotient rules, Boss Rush |
| 03 | **Function Anatomy Lab v3** | Curve Sketching | Interactive f/f'/f'' analysis, critical points, inflection |
| 04 | **Related Rates Shadow Lab v3** | Related Rates | Person + shadow + streetlamp, step-by-step guidance |
| 05 | **Optimization Paradox Lab v2** | Optimization | 6 classic problems, gut-choice failure system, slider exploration |
| 06 | **Integral Lockbox v5** | Integration Techniques | U-sub, By Parts, Partial Fractions, Trig Sub, 60+ problems |
| 07 | **Volumes of Revolution v3** | Solids of Revolution | Disk/washer/shell methods, 3D visualization |
| 08 | **Taylor Journey v0.9** | Taylor Series | "Close Enough" business game, Taylor vs Least Squares duel |
| 09 | **Zeno's Paradox** | Limits / Series Intro | Achilles & Tortoise, geometric series visualization |
| 10 | **Polar Explorer** | Polar Coordinates | Interactive r=f(θ) plotting, area calculation |
| 11 | **Vector Void v2** | Vectors | Space navigation game, dot/cross products, projections |

## 🎮 How to Use

1. Download and unzip the folder
2. Open any `.html` file in a web browser (Chrome, Firefox, Safari, Edge)
3. No installation or internet required - everything runs locally!

## 💡 Tips

- **Progress is saved** in browser localStorage for most tools
- **Keyboard shortcuts** are available (check each tool's help/info)
- **Teacher modes** available in some tools (Taylor Journey: hold Shift+~ for 2s)
- **Works offline** once downloaded
